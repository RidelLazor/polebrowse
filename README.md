## Features
- 🛡️ DNS-over-HTTPS protection (8 providers)
- 🌙 Dark & Light mode
- 📚 History, bookmarks, downloads
- 🔒 Safe browsing (dangerous site detection)
- 🦖 No internet page
- ⚡ Fast tab management

## Install

### Linux (all distros)
```bash
curl -fsSL https://raw.githubusercontent.com/RidelLazor/polebrowse/main/install.sh | bash
```
Or download and run:
```bash
bash install.sh
```

### Windows
Download `install.bat` and run as Administrator, or:
```cmd
install.bat
```

### macOS
```bash
bash install.sh
```

## Build from source
```bash
npm install
npm start              # dev mode
npm run make           # build packages
```

## Uninstall
```bash
bash install.sh --uninstall    # Linux/macOS
install.bat --uninstall        # Windows
```
